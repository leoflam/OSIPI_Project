Per utilizzare i tuoi file HTML, CSS e JavaScript in una pagina SharePoint, puoi seguire questi passaggi:

1. **Carica i file su SharePoint**:
   - Vai alla libreria documenti del tuo sito SharePoint.
   - Crea una nuova cartella (ad esempio, "Progetto") e carica i tuoi file `index.html`, `style.css` e `script.js` nelle rispettive sottocartelle (`css` e `javascript`).

2. **Ottieni i link ai file**:
   - Dopo aver caricato i file, ottieni i link diretti ai file `style.css` e `script.js`. Puoi farlo cliccando sui file e selezionando "Copia link".

3. **Crea una nuova pagina**:
   - Vai alla home page del tuo sito SharePoint e clicca su **+ Nuovo** > **Pagina**.

4. **Modifica la pagina**:
   - Clicca su **Modifica** in alto a destra per entrare in modalit√† di modifica.

5. **Aggiungi una web part di incorporamento**:
   - Posiziona il puntatore del mouse sopra o sotto una web part esistente o sotto l'area del titolo, clicca su **+** e seleziona la web part **Incorpora**.

6. **Incorpora il contenuto HTML**:
   - Nel riquadro delle propriet√† a destra, incolla il seguente codice HTML, modificando i percorsi dei file CSS e JavaScript con i link ottenuti al passaggio 2:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Progetto</title>
    <!-- Collegamento al file CSS -->
    <link rel="stylesheet" href="https://[your-sharepoint-site]/Progetto/css/style.css">
    <!-- Collegamento al file JavaScript -->
    <script src="https://[your-sharepoint-site]/Progetto/javascript/script.js" defer></script>
</head>
<body>
    <h1>Benvenuto nel mio progetto</h1>
    <p>Questo √® un esempio di pagina HTML con collegamenti a CSS e JavaScript.</p>
</body>
</html>
```

7. **Salva e pubblica**:
   - Dopo aver aggiunto il contenuto HTML, clicca su **Salva come bozza** o **Pubblica** per rendere la pagina visibile agli altri utenti del sito.

### Note
- **Sostituisci `[your-sharepoint-site]`** con l'URL del tuo sito SharePoint.
- Assicurati che i percorsi dei file CSS e JavaScript siano corretti e accessibili.
